pub struct Ty(_);

An object that represents a native system type

Each &Ty represents a basic system type, be it a primitive, a struct, a union, a pointer, or a function signature. The library uses this information to lay out values in memory.

Types are not attached to a context so they are reference-counted by LibJIT, so internally they are represented as Rc<Ty>. This represents a reference to the inner Ty.
